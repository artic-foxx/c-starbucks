Wander freely through a giant cafe with **16 interactive rooms**:

| Room | Activity |
| :---: | :--- |
| 👨‍🍳 **Kitchen** | Collect ingredients and craft drinks — Espresso, Latte, Caramel Macchiato, Mocha, Frappuccino & more! |
| 🧑‍💼 **Counter** | Serve pending orders to customers and earn stars |
| 🧙 **Magic** | Cast magical spells and mix potions |
| 🎊 **Party!** | Throw a party with decorations and fun |
| 🏴‍☠️ **Treasure** | Hunt for hidden treasures |
| 🐶 **Pets** | Play with adorable pets |
| 📚 **Books** | Browse the cafe library |
| 🪑 **Seating** | Chill out and relax |
| 🎨 **Art** | Paint and create art |
| 🏰 **Disney** | Sound effects and audio fun |
| 🌻 **Garden** | Grow and tend plants |
| 🎮 **Play** | Casual mini-games |
| 💃 **Dance** | Groove to the beat |
| 📸 **Photo** | Take pictures |
| 🎲 **Games** | Board games and puzzles |
| 🚻 **Restroom** | Keep the cafe clean |

### Gameplay

```
 WASD / Arrow Keys  ──►  Move your character
        E              ──►  Interact with rooms
        ESC            ──►  Go back to the main map
```

1. **Explore** — Walk around the open-world cafe map
2. **Enter rooms** — Stand near a room and press E
3. **Cook & Serve** — In the Kitchen, grab ingredients and craft drinks, then serve customers at the Counter
4. **Earn rewards** — Complete orders, collect stars, and level up
5. **Discover secrets** — Every room has something unique to try!

## Play

Just open `starbucks-game.html` in your browser — no build tools, no dependencies, no setup.

```bash
# That's it. Really.
open starbucks-game.html
```

## Tech Stack

- Pure **HTML** + **CSS** + **JavaScript** — zero dependencies
- Canvas-rendered pixel art sprites
- Web Audio API for in-game music & sound effects
